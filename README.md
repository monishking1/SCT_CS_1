# Task 1 - Caesar Cipher

## Description
This project is a simple Caesar Cipher program developed in Java as part of the SkillCraft Technology Cyber Security Internship.

The program allows users to:
- Enter a message
- Enter a shift value
- Encrypt the message
- Decrypt the encrypted message

## Features
- User-driven input
- Encryption using Caesar Cipher algorithm
- Decryption of encrypted text
- Supports uppercase and lowercase letters
- Preserves spaces and special characters

## Technologies Used
- Java
- IntelliJ IDEA

## Sample Output

Enter message: Hello World

Enter shift value: 3

Encrypted Text: Khoor Zruog

Decrypted Text: Hello World
